Color Usage Rules

  • Brass palette ONLY in the brand mark — never in the product UI.
  • Green (#16a34a) is used for UI accents and semantic truth indicators (not in the logo).
  • Slate 900 (#0f172a) bridges both palettes as primary text color.
  • Green = truth/verified. Red = lie/flagged. Amber = uncertain. Fixed semantics.

Typography Rules

  • Instrument Serif for the 'Verlo' wordmark (brass, 0.03em spacing, 0.8px stroke).
  • DM Sans 800 for page titles. DM Sans 700 for section heads, modal heads, card titles.
  • DM Sans 500 for body. DM Sans 600 for captions (uppercase, tracked 0.1em).
  • DM Mono for hex values, timestamps, code, technical labels, data displays.

Do's and Don'ts

Do

  • Use charcoal + brass for social/identity
  • Use slate + green palette for product UI
  • Maintain 25% mark-height clear space
  • Use Instrument Serif for the wordmark
  • Use DM Sans for all other brand comms
  • Keep wordmark single-color brass

Don't

  • Rotate, stretch, or distort the V mark
  • Use brass colors in the product UI
  • Mix typefaces within the wordmark
  • Use the mark below 24px minimum
  • Add shadows, gradients, or effects
  • Use the old geometric slab V
